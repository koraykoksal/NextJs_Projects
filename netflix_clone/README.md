# Netflix Clone App With NextJs



## Packages
- `firebase`
- `next`
- `postcss`
- `react`
- `react-dom`
- `react-toastify`
- `tailwindcss`


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
